🧠 APIs para stats de jugadores y partidas

Estas son las que sirven para hacer cosas tipo tracker de partidas.

1️⃣ HenrikDev API

La más usada en proyectos de comunidad.

Web / docs:
https://docs.henrikdev.xyz

KEY: HDEV-311dd021-864a-4c18-8845-b4c235f27e44


Qué te permite hacer
buscar cuentas

ver datos completos de una partida
leaderboard
cambios de rank
Ejemplos de endpoints

Account
/valorant/v1/account/{name}/{tag}

MMR
/valorant/v1/mmr/{region}/{name}/{tag}

Match history
/valorant/v3/matches/{region}/{name}/{tag}

Match details
/valorant/v4/match/{region}/{matchid}
Rate limit

Normalmente:
30 requests / minuto

👉 Es la mejor API pública para stats ahora mismo.
